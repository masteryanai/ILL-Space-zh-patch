**致开发者：** 如您能在后续更新中补全 ZH locres，社区的翻译内容可直接对接，欢迎联系协作。

### ⚠️ Known Limitations

#### DataTable Text Cannot Be Patched via locres

The following **766 entries** will **not display in Chinese** with the current patch:

| Namespace | Count | Content |
|-----------|-------|---------|
| `MissionStepsDataTable` | 232 | Tutorial NPC dialogue (e.g., all lines by Cara Armanteros) |
| `MissionObjectivesDatatable` | 256 | Mission objective text |
| `MissionsDataTable` | 102 | Mission descriptions |
| `HelperLinesDataTable` | 69 | Tutorial hints |
| `StarmapLocationTexts` | 50 | Starmap location descriptions |
| Other DataTable types | 57 | — |

**Root Cause:** In UE4, `FText` fields inside DataTables have their language lookup paths baked into the `.uasset` binary at compile time. The developers added these entries after the initial ZH release — they updated EN locres only, without establishing a ZH path in the binary. A locres patch cannot inject a lookup path that was never compiled in.

**Note to Developers:** If you can add ZH locres support for these entries in a future update, the community translations in this repo can be integrated directly. Feel free to reach out.
